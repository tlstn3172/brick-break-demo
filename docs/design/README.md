# Design Files

이 디렉토리는 Neon Breaker 게임의 UI/UX 디자인 파일을 포함합니다.

## 화면별 디자인

### 1. Start Screen (시작 화면)
- 경로: `./start-screen/`
- 파일:
  - `screen.png` - 디자인 스크린샷
  - `code.html` - HTML/CSS 구현 참조 코드

### 2. Gameplay Screen (게임 화면)
- 경로: `./gameplay-screen/`
- 파일:
  - `screen.png` - 디자인 스크린샷
  - `code.html` - HTML/CSS 구현 참조 코드

### 3. Game Over Screen (게임 오버 화면)
- 경로: `./gameover-screen/`
- 파일:
  - `screen.png` - 디자인 스크린샷
  - `code.html` - HTML/CSS 구현 참조 코드

## 디자인 시스템

### 컬러 팔레트
- **Primary**: `#0db9f2` (네온 시안)
- **Background Dark**: `#101e22`
- **Accent Purple**: `#a855f7`
- **Accent Teal**: `#2dd4bf`

### 타이포그래피
- **폰트**: Spline Sans
- **아이콘**: Material Symbols Outlined

### 디자인 특징
- 다크 모드 기반
- 네온 글로우 효과
- 부드러운 라운드 처리
- 그라데이션 및 블러 효과
