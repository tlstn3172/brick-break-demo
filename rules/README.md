# Project Rules Summary

이 디렉토리는 Neon Breaker 프로젝트의 개발 규칙을 정의합니다.

## 규칙 파일

### 1. [build-and-deployment.md](./build-and-deployment.md)
빌드 및 배포 전략을 정의합니다.

- **빌드**: GitHub Actions 사용
- **배포**: GitHub Pages 사용
- 자동화된 CI/CD 파이프라인

### 2. [development-principles.md](./development-principles.md)
개발 원칙 및 코드 품질 가이드라인을 정의합니다.

- **TDD**: 코어 로직에만 적용 (UI 제외)
- **SOLID 원칙**: 모든 코드에 적용
- 코드 품질 가이드라인 및 체크리스트

## 규칙 준수

모든 개발자는 이 규칙을 따라야 하며, 코드 리뷰 시 이 규칙을 기준으로 검토합니다.
